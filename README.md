关注这个数据三年左右，不要问为什么这个指数于2016年底中断了！

* 房天下发布于每月的最后一天发布，或者次月的第一天发布，先于官方数据发布（总是会在这一两天非常积极的去查看它的报告）
* 数据经常与官方数据有一定出入
* 在政府面临着空前压力的情况下，透露甚至夸大市场的疯狂，在政府绞尽脑汁调控市场时，
